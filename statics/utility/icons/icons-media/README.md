# icons-media

Les icônes fonctionnelles sont des symboles visuels qui accompagnent l’utilisateur dans ses actions et qui aident à sa compréhension de l’interface.

## Dépendances
```shell
icons-media
└─ core
```

## Utilisation
Afin d’utiliser le composant `icons-media`, il est nécessaire d’ajouter les fichiers de styles et de scripts présents dans le dossier dist dans l'ordre suivant :
```html
<html>
  <head>
    <link href="css/core/core.min.css" rel="stylesheet">
    <link href="css/icons-media/icons-media.min.css" rel="stylesheet">
  </head>
  <body>
  </body>
</html>
```

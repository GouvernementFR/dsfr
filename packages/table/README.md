# @gouvfr/table

Les tableaux sont utilisés pour organiser et afficher les informations d'un jeu de données.

## Dépendances
```shell
table
└─ schemes
└─ core
└─ typography
```

## Installation
Installation via NPM:
```
npm install @gouvfr/table
```
Via Yarn :
```
yarn add install @gouvfr/table
```

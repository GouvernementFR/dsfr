# @gouvfr/toggle

Le toggle est un bouton permettant de passer d'un état activé à un état désactivé.

## Dépendances
```shell
toggle
└─ schemes
└─ core
└─ icons
└─ forms
└─ typography
```

## Installation
Installation via NPM:
```
npm install @gouvfr/toggle
```
Via Yarn :
```
yarn add install @gouvfr/toggle
```

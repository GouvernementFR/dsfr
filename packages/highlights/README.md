# @gouvfr/highlights

La mise en exergue permet à l’utilisateur de distinguer et repérer une information facilement.

## Dépendances
```shell
highlights
└─ schemes
└─ core
└─ body
```

## Installation
Installation via NPM:
```
npm install @gouvfr/highlights
```
Via Yarn :
```
yarn add install @gouvfr/highlights
```

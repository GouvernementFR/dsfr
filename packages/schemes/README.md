# @gouvfr/schemes

Les color schemes permettent la définition de couleurs principales et du darkmode pour le Design System de l'Etat.

## Dépendances
```shell
schemes
└─ core
```

## Installation
Installation via NPM:
```
npm install @gouvfr/schemes
```
Via Yarn :
```
yarn add install @gouvfr/schemes
```

# @gouvfr/radios

Les boutons radios permettent à l’utilisateur de sélectionner une seule option dans une liste.

## Dépendances
```shell
radios
└─ schemes
└─ core
└─ forms
```

## Installation
Installation via NPM:
```
npm install @gouvfr/radios
```
Via Yarn :
```
yarn add install @gouvfr/radios
```

# @gouvfr/pagination

La pagination permet à l’utilisateur de naviguer entre les différentes pages d’une liste d'éléments.

## Dépendances
```shell
pagination
└─ schemes
└─ core
└─ icons
└─ body
```

## Installation
Installation via NPM:
```
npm install @gouvfr/pagination
```
Via Yarn :
```
yarn add install @gouvfr/pagination
```
